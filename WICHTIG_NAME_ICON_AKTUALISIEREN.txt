WICHTIG – falls Name und Icon sich bisher nicht geändert haben

Du musst sehr wahrscheinlich die bestehende installierte App zuerst vom Handy löschen
und danach das Android-Projekt neu synchronisieren und neu bauen.

Bitte genau diese Reihenfolge:

1. ZIP komplett in einen NEUEN Ordner entpacken
2. Im neuen Ordner ausführen:
   npm install
   npm run build
   npx cap sync android
3. Android Studio öffnen:
   npx cap open android
4. In Android Studio:
   Build > Clean Project
   Build > Rebuild Project
5. Die alte App auf dem Handy DEINSTALLIEREN
6. Neue APK installieren

Geänderte Dateien:
- android/app/src/main/res/values/strings.xml
- android/app/src/main/AndroidManifest.xml
- android/app/src/main/res/mipmap-*/ic_launcher*.png
- android/app/src/main/res/mipmap-anydpi-v26/ic_launcher*.xml
- capacitor.config.ts
- src/main.js
